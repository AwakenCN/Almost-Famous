```
    大风起兮云飞扬，力拔山兮气盖世。
    一曲芳华，一首挽歌。
    有道是天道好轮回，苍天饶过谁。
    漫漫长路，唯有尔等与吾同行，喜也，幸也。
    俱往矣，数风流人物还看今朝。
```
